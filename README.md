# Release Notes Compiler

## Exercise Goal
Learn to use VS Code's split view and Markdown preview while combining multiple files.

## Instructions
1. Clone this repository
2. Open the three text files: `features.txt`, `bugs.txt`, and `credits.txt`
3. Use split view to see them side-by-side
4. Create a new file `v1.0_release_notes.md`
5. Open "Preview to the Side" for the markdown file
6. Combine the information from the text files into a formatted release notes document
7. Use Markdown formatting: headers, bold text, bullet points
8. Commit and push your work

## Skills Practiced
- Multi-window editing
- Live Markdown preview
- Formatting documentation
- Git workflow in VS Code
